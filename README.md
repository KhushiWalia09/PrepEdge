Interview Preparation Platform built with Flask

## Features
- User authentication (Login & Signup)
- Role-based access (Admin / Aspirant)
- Mock Interviews with random questions
- Question Bank with category filter
- Admin panel to add interview questions
- Session-based authentication

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS

## Roles
- **Admin**: Can add questions
- **Aspirant**: Can practice interviews

## How to Run
1. Create virtual environment
2. Install dependencies
3. Run `python app.py`

## Future Improvements
- Timed mock interviews
- Progress tracking
